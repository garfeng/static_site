# Static Site

**一个纯静态（？）的网站程序**

## 目的

节省流量/内存/空间，响应速度快，呵呵呵呵。

界面和数据分离，可以放在不同的域名（需要数据服务器允许跨域访问）

## 缺点

搜索引擎一个字都不会收录，请慎用。
